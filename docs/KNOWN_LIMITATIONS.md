# Known Limitations

* V1 is focused on the core working pipeline and release-safe packaging.
* Best results depend heavily on the quality of the source rig and blendshape naming.
* Some facial libraries and character-specific presets still need additional polish.
* Broader compatibility smoothing is still future work.
* Advanced realism for jaw, teeth, tongue, and subtle mouth behavior varies by source character quality.
* External dependencies such as local models, Whisper, and Piper are not bundled as a universal one-click install in V1.
